# BinaryTreeApp



BinaryTreeApp — это десктопное приложение на C# (.NET 9.0) для работы с различными типами бинарных деревьев.  

Оно позволяет создавать, редактировать и визуализировать деревья, выполнять обходы, подсчитывать максимумы в листьях и многое другое.  



Проект предназначен для **образовательных целей**, демонстрируя использование паттернов проектирования (Facade, Factory, Strategy), generic-типов и WinForms для UI.



## Основные возможности



- **Типы деревьев**:

&nbsp; - Binary Search Tree (BST) — упорядоченное бинарное дерево поиска

&nbsp; - Balanced Tree — сбалансированное дерево (например, AVL или Red-Black)

&nbsp; - Simple Tree — простое бинарное дерево без упорядочения



- **Операции**:

&nbsp; - Вставка и удаление элементов

&nbsp; - Поиск минимума/максимума (для упорядоченных деревьев)

&nbsp; - Обходы: In-Order, Pre-Order, Post-Order, Level-Order

&nbsp; - Подсчёт количества максимумов в листьях дерева

&nbsp; - Получение значений на заданной глубине

&nbsp; - Очистка дерева



- **Визуализация**:

&nbsp; - Адаптивная отрисовка дерева в панели (масштабируется под размер окна)



- **Ввод данных**:

&nbsp; - Из строки с числами (разделёнными пробелами или запятыми) — последовательная вставка

&nbsp; - Из скобочной нотации для Simple Tree (например: `8(1,2(3,4))`) — явное задание структуры



- **UI**:

&nbsp; - Выбор типа дерева через радиокнопки

&nbsp; - Ввод строки и кнопка «Выполнить»

&nbsp; - Панель информации с полной статистикой (узлы, высота, мин/макс, обходы, максимумы в листьях)



## Требования



- **ОС**: Windows (из-за WinForms)

- **.NET**: SDK 9.0 или выше

- **IDE**: Visual Studio 2022+ (рекомендуется)



## Установка и запуск



1. Клонируйте репозиторий



```bash

git clone https://github.com/ВашНик/BinaryTreeApp.git

cd BinaryTreeApp

```



2. Откройте проект в Visual Studio  

&nbsp;  → Откройте файл `BinaryTreeApp.sln` или `BinaryTreeApp.csproj`



3. Соберите и запустите



&nbsp;  - В Visual Studio: Build → Build Solution → Debug → Start Debugging (F5)

&nbsp;  - Или из командной строки:



```bash

dotnet build

dotnet run

```



## Как использовать



1. Выберите тип дерева через радиокнопки (BinarySearch / Balanced / Simple)

2. Введите данные:

&nbsp;  - Простой список: `5 3 7 1 4 9`

&nbsp;  - Структура для Simple-дерева: `8(1,2(3,4))` или `10(,6(4,8))`

3. Нажмите «Выполнить»

4. Смотрите дерево слева, статистику справа

5. Очистите дерево кнопкой «Очистить»



## Архитектура проекта



- **Facades** → `TreeFacade<T>` — единый интерфейс для UI

- **Factories** → `TreeFactory<T>` — создание нужного типа дерева

- **Models** → `ITree<T>`, `ITreeNode<T>` — базовые интерфейсы

- **Services** → `TreeDrawer<T>` (визуализация), `TreeOperations<T>` (операции)

- **Strategies** → паттерн Strategy для всех видов обходов + `LeafTraversalStrategy`

- **UI** → `MainForm` на Windows Forms
